Its written in Node JS using the Botkit framework. Please give it a download or use this docker image and run it by filling out this docker-compose.yml

version: '2'
services:
  pvr-slackbot:
    restart: always
    image: siebes/pvr-slackbot:latest
    environment:
    - "SLACK_TOKEN="
    - "SONARR_KEY="
    - "SONARR_HOST="
    - "SONARR_PORT="
    - "CP_KEY="
    - "CP_HOST="
    - "CP_PORT="

Any feedback/bug reports can be done via the github page. Pull requests are also welcome.
